Untuk dapat melakukan pembayaran S88pay melalui API, operator harus membuat URL yang berisi token dan parameter. Anda dapat melihat dokumentasi **fungsi API** untuk URL dan parameter dengan jelas.

Berdasarkan keamanan pada prasyarat, parameter biasanya menggunakan kunci dengan fungsi enkripsi. Anda dapat melihat [enkripsi / dekripsi](/api-reference/encrypt-decrypt).

## Membuat Permintaan

Misalkan kita ingin membuat permintaan pembayaran, enpoint haruslah dopayment. Dan kita perhatikan parameter yang diperlukan

<Steps>

### Gabungkan semua parameter menjadi satu string, dipisahkan oleh karakter `&`

misalnya, untuk membuat permintaan dalam layanan kami.

```php
$str = merchant_code=xxx&merchant_api_key=xxx&transaction_code=xxx&transaction_timestamp=xxx&payment_code=xxx&transaction_amount=xxx&user_id=xxx&currency_code=xxx
```

### Enkripsi menggunakan encrypt_decrypt

Pelajari lebih lanjut tentang [encrypt_decrypt ](/api-reference/encrypt-decrypt)

Contoh Enkripsi:

```php
$key = encrypt_decrypt('encrypt', $str, '{your_api_key}', '{your_secret_key}')
```
