Dokumen ini bertujuan untuk menyediakan spesifikasi teknis komprehensif untuk mengintegrasikan dan melakukan pengujian integrasi dengan Gateway Pembayaran S88pay. Ini mencakup fungsi, parameter, format hasil, dan tanggapan kesalahan, memastikan proses integrasi yang lancar dengan kompleksitas kode yang minimal.

## Istilah & Definisi

Agar dapat membaca dokumentasi dengan jelas, kita perlu memiliki persepsi yang sama. Istilah dan definisi disediakan di bawah ini untuk memudahkan pembacaan.

| Istilah             | Deskripsi                                                                                                      |
| ------------------- | -------------------------------------------------------------------------------------------------------------- |
| Operator            | Mitra yang ingin mengintegrasikan dengan Gateway Pembayaran S88Pay                                             |
| Penyedia Pembayaran | Penyedia API Pembayaran untuk Operator                                                                         |
| Situs BO            | Situs backoffice, didedikasikan untuk Operator untuk memverifikasi transaksi.                                  |
| Situs Anggota       | Situs web yang dapat diakses oleh anggota untuk masuk dan memulai transaksi melalui Gateway Pembayaran S88pay. |
| Pengguna / Anggota  | Pelanggan yang memulai transaksi melalui Gateway Pembayaran S88pay.                                            |

Jika Anda tidak jelas, silakan lanjutkan untuk menghubungi tim dukungan kami.
